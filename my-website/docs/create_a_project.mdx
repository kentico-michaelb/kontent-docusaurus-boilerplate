#### A "Project" is the primary organizational unit of your content.

To create a new project:

1.  Click the project switcher in the top left.
2.  Click Projects.
3.  Click Create new project in the top-right corner.
4.  Type a name for your project.
5.  Click Create project.
6.  Click Open project.

:::tip Pro Tip...

Reviewing the [official Kontent Documentation](https://docs.kontent.ai/ "Kontent Documentation") will allow you better understand important concepts like [Content Modeling](https://docs.kontent.ai/tutorials/set-up-kontent/content-modeling/what-is-content-modeling "Kontent Content Modeling"), [Terminology](https://docs.kontent.ai/tutorials/assorted-items/dive-deeper-into-kentico-kontent/terminology "Kontent Terminology"), and general [Project Setup ](https://docs.kontent.ai/tutorials/set-up-kontent/set-up-your-project/overview "Set up your project")concepts.

:::
